---
description: Як завантажити проект у чистий репозиторій GitHub
---

Щоб завантажити цей проект у новий репозиторій, виконайте ці кроки у терміналі (переконайтеся, що ви знаходитесь у папці `swimming-events`):

1. Ініціалізуйте локальний репозиторій:
```bash
git init
```

2. Додайте всі файли до індексу:
```bash
git add .
```

3. Зробіть перший коміт:
```bash
git commit -m "Initial commit: Swimming Events with Redux and Charts"
```

4. Перейменуйте головну гілку (якщо потрібно):
```bash
git branch -M main
```

5. Створіть чистий репозиторій на GitHub (або іншому сервісі) та скопіюйте його URL.

6. Прив'яжіть локальний репозиторій до віддаленого (замініть `<URL>` на ваше посилання):
```bash
git remote add origin <URL>
```

7. Завантажте код на сервер:
```bash
git push -u origin main
```
