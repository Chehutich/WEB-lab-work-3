# Swimming Events - Реєстрація на змагання

Простий та зручний застосунок для керування реєстрацією учасників на змагання з плавання, побудований на React та Redux Toolkit.

## Основні можливості
- **Керування подіями**: Перегляд доступних змагань (Home page).
- **Реєстрація**: Форма реєстрації з валідацією (Zod + React Hook Form). Мінімальний вік — 12 років.
- **Список учасників**: Перегляд зареєстрованих осіб для кожної події з можливістю пошуку.
- **Візуалізація**: Інтерактивний графік активності реєстрацій за днями (Recharts).
- **Імпорт даних**: Можливість імпортувати учасників із зовнішнього джерела (JSONPlaceholder).
- **Темна тема**: Підтримка темного та світлого режимів із збереженням вашого вибору.
- **Локальне сховище**: Всі дані зберігаються у `localStorage`, тому вони не зникають після оновлення сторінки.

## Технології
- **Frontend**: React 19, Vite, TypeScript.
- **State Management**: Redux Toolkit (Slices, Entity Adapter, Thunks).
- **Validation**: Zod.
- **Icons**: Lucide React.
- **Styling**: Vanilla CSS.

## Як запустити проект локально
1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть сервер для розробки:
   ```bash
   npm run dev
   ```
3. Відкрийте [http://localhost:5173](http://localhost:5173) у вашому браузері.
