# Swimming Events - React + Redux Toolkit

Проєкт для керування реєстрацією учасників на змагання з плавання.

## Основні функції
- **Глобальний стейт**: Використання Redux Toolkit для керування подіями, учасниками та темою.
- **Візуалізація**: Графік активності реєстрацій за допомогою `Recharts`.
- **Зовнішній Імпорт**: Функція імпорту учасників із зовнішнього API (JSONPlaceholder).
- **Темна Тема**: Підтримка темного та світлого режимів із збереженням у `localStorage`.
- **Валідація**: Використання `Zod` для валідації форм реєстрації (обмеження віку 18+).
- **Адаптивний дизайн**: Коректне відображення на мобільних пристроях.

## Технологічний стек
- React 19
- Redux Toolkit (@reduxjs/toolkit)
- React Router 7
- Zod + React Hook Form
- Lucide React (іконки)
- Recharts (графіки)
- React Hot Toast (сповіщення)

## Як запустити
1. Клонувати репозиторій.
2. Встановити залежності:
   ```bash
   npm install
   ```
3. Запустити в режимі розробки:
   ```bash
   npm run dev
   ```
